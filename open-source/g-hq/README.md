## Instructions

#### To run locally:
- Fork and clone the repo
- In command line, `git clone {insert-your-repo-SSH-key}`
- Run `npm install`
- Run `brew install yarn`
- In order to view on your local host, run `yarn start` (or `npm start` if yarn doesn't work)
